# Contributing

This is a community resource. Contributions are welcome — whether it's a device-specific migration procedure, an improvement to the Claude prompt, or a correction.

---

## Ways to Contribute

### Add a Device Guide
If you've migrated a device type not covered in the `devices/` folder, add a guide. Use the existing guides as templates.

### Improve the Prompt
If you've found that the Claude prompt works better with modifications for your setup, share it. Especially useful:
- Variations for specific device types
- Prompt additions for capturing settings we don't currently capture
- Improvements for non-English HA setups

### Report Issues
If something in the guide is incorrect, outdated, or unclear, open an issue.

### Share Your Results
If you used this guide for your migration, sharing a summary of your setup (device count, coordinator hardware, anything that didn't work as expected) helps improve the guide for others.

---

## Guidelines

- Keep device guides focused and practical — specific steps, not vague advice
- Test before contributing — ideally you've done the migration yourself
- Don't include personal information (entity IDs from your home, etc.)
- Keep the prompt generic — it should work for any ZHA setup, not just specific hardware

---

## Pull Requests

Fork the repo, make your changes, and open a PR with a brief description of what you added or changed. No formal requirements — just make it clear what you're contributing and why.
